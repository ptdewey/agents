## Phase 1: Research & Setup
- [ ] Read Pi extension documentation (`docs/extensions.md` and `examples/extensions/`).
- [ ] Initialize a new Node.js project (`npm init -y`) or use the Pi extension template.
- [ ] Set up TypeScript (optional but recommended).

## Phase 2: Explore the Pi SDK & Lifecycle Events
- [ ] Review `docs/sdk.md` for agent lifecycle hooks.
- [ ] Identify event/hook for agent finish (e.g., `onRunComplete`).
- [ ] Identify event/hook for waiting on user input (e.g., `onUserInputRequest`).
- [ ] Write a basic extension script with console logs to verify hooks.

## Phase 3: Implement OS Notifications
- [ ] Research cross-platform Node.js notification libraries (e.g., `node-notifier`).
- [ ] Install the chosen library.
- [ ] Write a standalone test script for OS notifications.

## Phase 4: Integration
- [ ] Integrate notification logic into Pi extension lifecycle hooks.
- [ ] Define "Task Completed" notification payload.
- [ ] Define "Waiting for Input" notification payload.

## Phase 5: Testing & Polish
- [ ] Load the extension into a local Pi session.
- [ ] Test finish notification with a simple task.
- [ ] Test input notification with a task needing clarification.
- [ ] Add config options (sounds, custom messages).
- [ ] Write `README.md`.
